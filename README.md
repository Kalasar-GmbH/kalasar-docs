# Kalasar Docs

Shared knowledge base for topics that span across Sales, Onboarding, and Development.

This repo contains research, guides, and reference material that any team member or AI agent working on Kalasar needs to understand — regardless of whether they're building features, onboarding a customer, or explaining the product.

## Structure

- `meta-topics/` — Meta Ads platform: CAPI, Conversion Leads, datasets, pixels, optimization
- `crm-topics/` — CRM platforms: Attio data model, HubSpot, Salesforce integration patterns

## When to add something here

Add a document when:
- The topic is relevant across multiple functions (not just dev, not just sales)
- The information was hard to find or piece together from official docs
- Future team members or agents would otherwise have to re-research it
